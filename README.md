# Decoding the 2026 Tamil Nadu Assembly Election
Codebasics Resume Project Challenge

## Three Stories
1. The Flip Story — 163 of 234 constituencies had a new winner
2. The Vote Share Story — TVK debuted at 35.1%, ahead of DMK and AIADMK
3. The Margin Story — Average winning margin fell 43%

## Data Sources
- tn_2021_results.csv — Trivedi Centre for Political Data (Ashoka University)
- tn_2026_results.csv — Election Commission of India (results.eci.gov.in)
- constituency_master.csv — ECI constituency master list

## Files
- tn_election_analysis.ipynb — Full analysis notebook (run in Google Colab)

## How to Run
1. Open tn_election_analysis.ipynb in Google Colab
2. Upload the 3 CSV files
3. Run all cells top to bottom

## Disclaimer
Non-partisan analysis using only public ECI data.
